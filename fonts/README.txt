KFGQPC Uthmanic Script HAFS — Required Font Files
==================================================

Place the following files in this directory:

  KFGQPCUthmanicScriptHAFS.woff2   ← primary (best performance)
  KFGQPCUthmanicScriptHAFS.ttf     ← fallback

Download from:
  https://cdn.jsdelivr.net/npm/kfgqpc-uthmanic-script-hafs-regular@1.0.0/

Or search: "KFGQPC Uthmanic Script HAFS Regular" on arabicfonts.net

License: Free for use — King Fahd Glorious Quran Printing Complex (KFGQPC)
ISBN: 978-603-8010-15-0

Until the font file is present, the app falls back to Noto Naskh Arabic.
